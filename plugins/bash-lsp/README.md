# Bash LSP Plugin

Bash Language Server integration for Claude Code.

## Installation

Install bash-language-server:

```bash
npm install -g bash-language-server
```

## Features

- Autocomplete for Bash scripts
- Diagnostics (syntax errors, warnings)
- Go to definition
- Find references
- Hover information

## Usage

LSP features are automatically available for `.sh` and `.bash` files.
